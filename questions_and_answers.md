###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>C</b></summary>
<p>

#### Answer: ?

<i>As the first value of nums array was zero, if statement considered it as falsy, for that reason, the first iteration was skipped and count was not increaed. For that reason, the final value of count was printed 3 instead of 4.  </i>

</p>
